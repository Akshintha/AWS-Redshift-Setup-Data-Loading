# AWS Redshift Setup & Data Loading
This project demonstrates the basic steps to set up an Amazon Redshift cluster and load data into it. It covers creating a cluster, connecting to Redshift, creating tables, and importing data for analysis.

## What I Learned
- Create an Amazon Redshift cluster
- Configure IAM roles
- Connect to Redshift Query Editor
- Create a database and tables
- Load data from Amazon S3 into Redshift
- Query data using SQL
